What is Ansible?

Ansible is an open-source automation tool used for configuration management, application deployment, and orchestration. It uses SSH and YAML-based playbooks, making it simple and powerful.

Ansible Architecture

  • Control Node: Machine where Ansible is installed
  • Managed Nodes: Servers managed by Ansible
  • Inventory: List of managed nodes
  • Playbooks: YAML files defining tasks

Inventory Example

[workers]
worker1 ansible_host=10.0.0.1
worker2 ansible_host=10.0.0.2

[all:vars]
ansible_user=ubuntu

Playbook Example

- name: Install nginx
  hosts: workers
  become: true
  tasks:
    - name: Install nginx
      apt:
        name: nginx
        state: present

Common Ansible Commands

  • ansible all -m ping
  • ansible-playbook playbook.yml
  • ansible-inventory --list
